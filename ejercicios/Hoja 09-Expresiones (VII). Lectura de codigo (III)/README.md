# Ejercicios lectura de código.
El objetivo es interpretar las líneas de código y comprender la salida del console.log como resultado.

## Ejercicio 1

```javascript
let edad = 20;
let tieneCarnet = true;
console.log(edad >= 18 && tieneCarnet); 
```

## Ejercicio 2
